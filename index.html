CONQUER

Simple NON-Bootstrap Template

NOT Bootstrap v3.3.6

I used no Bootstrap for this page. This was a pure HTML/CSS/ Javascript construction. I may try this again at a later date using Bootstrap as well to test my Bootstrap skills. But for my first project I wanted to do the whole thing 'by hand'.

Responsive Design

Conquer Template was originally described by TemplateMo. On their website, it was stated that their site could be used for any kind of website, with no credit link required. As a result, I have used it as a practice design project. All images by Aaron. Thank you for visiting my website. Please come again.

Parallax Layout

No one should ever try to do a parallax effect with pure Javascript. Well, I suppose masochists should. But given that it literally takes like two lines of code to do with Javascript, while with CSS it takes about 30 lines, I don't know why anyone would really bother unless they were purists in the worst sense of the word.
